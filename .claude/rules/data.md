# Data Rules

## Structure

- 모든 데이터는 `data/` 디렉터리에 둔다
- 원본 데이터는 `data/raw/`에 저장
- 가공 데이터는 `data/processed/`에 저장

## Rules

- 원본 데이터는 절대 수정하지 않는다
- 항상 raw → processed 흐름 유지
- 코드에서 raw 데이터를 직접 수정 금지
- 데이터 경로 하드코딩 금지 (상대경로 사용)

## Processing

- 전처리는 재현 가능해야 한다
- 동일 입력 → 동일 출력 보장
- 임시 수작업 수정 금지
- 전처리 로직은 코드로 명시

## Validation

- 중복 제거 필수
- 결측값 처리 명시
- 이상 데이터 필터링

## Output

- 결과 데이터는 명확한 파일명 사용
- 기존 파일 덮어쓰기 금지
